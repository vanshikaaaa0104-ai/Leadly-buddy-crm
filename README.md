# Leadly Buddy CRM

## Description
Leadly Buddy CRM is a lead management system that helps users track and manage leads through different stages of a sales pipeline.

## Features
- User authentication (Google and email/password)
- Add new leads
- Lead status tracking
- Dashboard statistics
- Search leads by name, email, or company
- Lead pipeline management
